UnsafeCell, but Sync.

This is just an UnsafeCell, except it implements Sync if T implements Sync.

UnsafeCell doesnâ€™t implement Sync, to prevent accidental mis-use. You can use SyncUnsafeCell instead of UnsafeCell to allow it to be shared between threads, if thatâ€™s intentional. Providing proper synchronization is still the task of the user, making this type just as unsafe to use.
