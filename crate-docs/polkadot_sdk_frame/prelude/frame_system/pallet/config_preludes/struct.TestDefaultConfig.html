Struct polkadot_sdk_frame::prelude::frame_system::pallet::config_preludes::TestDefaultConfig

pub struct TestDefaultConfig;
Expand description

Provides a viable default config that can be used with derive_impl to derive a testing pallet config based on this one.

See Test in the default-config example pallet’s test.rs for an example of a downstream user of this particular TestDefaultConfig
