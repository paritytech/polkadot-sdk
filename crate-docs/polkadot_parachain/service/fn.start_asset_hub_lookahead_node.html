Start a shell node which should later transition into an Aura powered parachain node. Asset Hub uses this because at genesis, Asset Hub was on the shell runtime which didn’t have Aura and needs to sync and upgrade before it can run AuraApi functions.

Uses the lookahead collator to support async backing.
