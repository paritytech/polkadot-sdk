Expand description

A request to the Runtime API subsystem.

Variants§

§

Version(Sender<Result<u32, RuntimeApiError>>)

Get the version of the runtime API, if any.

§

Authorities(Sender<Result<Vec<Public, Global>, RuntimeApiError>>)

Get the next, current and some previous authority discovery set deduplicated.

§

Validators(Sender<Result<Vec<Public, Global>, RuntimeApiError>>)

Get the current validator set.

§

ValidatorGroups(Sender<Result<(Vec<Vec<ValidatorIndex, Global>, Global>, GroupRotationInfo<u32>), RuntimeApiError>>)

Get the validator groups and group rotation info.

§

AvailabilityCores(Sender<Result<Vec<CoreState<H256, u32>, Global>, RuntimeApiError>>)

Get information on all availability cores.

§

PersistedValidationData(Id, OccupiedCoreAssumption, Sender<Result<Option<PersistedValidationData<H256, u32>>, RuntimeApiError>>)

Get the persisted validation data for a particular para, taking the given OccupiedCoreAssumption, which will inform on how the validation data should be computed if the para currently occupies a core.

§

AssumedValidationData(Id, H256, Sender<Result<Option<(PersistedValidationData<H256, u32>, ValidationCodeHash)>, RuntimeApiError>>)

Get the persisted validation data for a particular para along with the current validation code hash, matching the data hash against an expected one.

§

CheckValidationOutputs(Id, CandidateCommitments<u32>, Sender<Result<bool, RuntimeApiError>>)

Sends back true if the validation outputs pass all acceptance criteria checks.

§

SessionIndexForChild(Sender<Result<u32, RuntimeApiError>>)

Get the session index that a child of the block will have.

§

ValidationCode(Id, OccupiedCoreAssumption, Sender<Result<Option<ValidationCode>, RuntimeApiError>>)

Get the validation code for a para, taking the given OccupiedCoreAssumption, which will inform on how the validation data should be computed if the para currently occupies a core.

§

ValidationCodeByHash(ValidationCodeHash, Sender<Result<Option<ValidationCode>, RuntimeApiError>>)

Get validation code by its hash, either past, current or future code can be returned, as long as state is still available.

§

CandidatePendingAvailability(Id, Sender<Result<Option<CommittedCandidateReceipt<H256>>, RuntimeApiError>>)

Get a the candidate pending availability for a particular parachain by parachain / core index

§

CandidateEvents(Sender<Result<Vec<CandidateEvent<H256>, Global>, RuntimeApiError>>)

Get all events concerning candidates (backing, inclusion, time-out) in the parent of the block in whose state this request is executed.

§

SessionExecutorParams(u32, Sender<Result<Option<ExecutorParams>, RuntimeApiError>>)

Get the execution environment parameter set by session index

§

SessionInfo(u32, Sender<Result<Option<SessionInfo>, RuntimeApiError>>)

Get the session info for the given session, if stored.

§

DmqContents(Id, Sender<Result<Vec<InboundDownwardMessage<u32>, Global>, RuntimeApiError>>)

Get all the pending inbound messages in the downward message queue for a para.

§

InboundHrmpChannelsContents(Id, Sender<Result<BTreeMap<Id, Vec<InboundHrmpMessage<u32>, Global>, Global>, RuntimeApiError>>)

Get the contents of all channels addressed to the given recipient. Channels that have no messages in them are also included.

§

CurrentBabeEpoch(Sender<Result<Epoch, RuntimeApiError>>)

Get information about the BABE epoch the block was included in.

§

FetchOnChainVotes(Sender<Result<Option<ScrapedOnChainVotes<H256>>, RuntimeApiError>>)

Get all disputes in relation to a relay parent.

§

SubmitPvfCheckStatement(PvfCheckStatement, Signature, Sender<Result<(), RuntimeApiError>>)

Submits a PVF pre-checking statement into the transaction pool.

§

PvfsRequirePrecheck(Sender<Result<Vec<ValidationCodeHash, Global>, RuntimeApiError>>)

Returns code hashes of PVFs that require pre-checking by validators in the active set.

§

ValidationCodeHash(Id, OccupiedCoreAssumption, Sender<Result<Option<ValidationCodeHash>, RuntimeApiError>>)

Get the validation code used by the specified para, taking the given OccupiedCoreAssumption, which will inform on how the validation data should be computed if the para currently occupies a core.

§

Disputes(Sender<Result<Vec<(u32, CandidateHash, DisputeState<u32>), Global>, RuntimeApiError>>)

Returns all on-chain disputes at given block number. Available in v3.

§

UnappliedSlashes(Sender<Result<Vec<(u32, CandidateHash, PendingSlashes), Global>, RuntimeApiError>>)

Returns a list of validators that lost a past session dispute and need to be slashed. V5

§

KeyOwnershipProof(Public, Sender<Result<Option<OpaqueKeyOwnershipProof>, RuntimeApiError>>)

Returns a merkle proof of a validator session key. V5

§

SubmitReportDisputeLost(DisputeProof, OpaqueKeyOwnershipProof, Sender<Result<Option<()>, RuntimeApiError>>)

Submits an unsigned extrinsic to slash validator who lost a past session dispute. V5

§

MinimumBackingVotes(u32, Sender<Result<u32, RuntimeApiError>>)

Get the minimum required backing votes.

§

DisabledValidators(Sender<Result<Vec<ValidatorIndex, Global>, RuntimeApiError>>)

Returns all disabled validators at a given block height.

§

ParaBackingState(Id, Sender<Result<Option<BackingState<H256, u32>>, RuntimeApiError>>)

Get the backing state of the given para.

§

AsyncBackingParams(Sender<Result<AsyncBackingParams, RuntimeApiError>>)

Get candidate’s acceptance limitations for asynchronous backing for a relay parent.

If it’s not supported by the Runtime, the async backing is said to be disabled.
