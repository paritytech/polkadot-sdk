Ensures that the value of PriorityBoostPerHeader matches the value of tip_boost_per_header.

We want two transactions, TX1 with N headers and TX2 with N+1 headers, have almost the same priority if we’ll add tip_boost_per_header tip to the TX1. We want to be sure that if we add plain PriorityBoostPerHeader priority to TX1, the priority will be close to TX2 as well.
