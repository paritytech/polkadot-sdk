Module bridge_runtime_common::refund_relayer_extension

source ·
Expand description

Signed extension that refunds relayer if he has delivered some new messages. It also refunds transaction cost if the transaction is an utility.batchAll() with calls that are: delivering new messsage and all necessary underlying headers (parachain or relay chain).
