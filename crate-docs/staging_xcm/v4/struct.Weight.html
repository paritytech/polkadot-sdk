pub const fn checked_div_per_component(self, other: &Weight) -> Option<u64>

Calculates how many other fit into self.

Divides each component of self against the same component of other. Returns the minimum of all those divisions. Returns None in case all components of other are zero.

This returns Some even if some components of other are zero as long as there is at least one non-zero component in other. The division for this particular component will then yield the maximum value (e.g u64::MAX). This is because we assume not every operation and hence each Weight will necessarily use each resource.
