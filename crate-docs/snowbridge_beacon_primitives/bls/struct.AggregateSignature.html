Struct snowbridge_beacon_primitives::bls::AggregateSignature

pub struct AggregateSignature {
    pub point: ECP2,
}
Expand description

Allows for the adding/combining of multiple BLS Signatures.

This may be verified against some AggregatePublicKey.
