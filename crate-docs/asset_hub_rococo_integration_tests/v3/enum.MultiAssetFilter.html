source · 
pub enum MultiAssetFilter {
    Definite(MultiAssets),
    Wild(WildMultiAsset),
}
Expand description

MultiAsset collection, defined either by a number of MultiAssets or a single wildcard.

Variants§

§

Definite(MultiAssets)

Specify the filter as being everything contained by the given MultiAssets inner.

§

Wild(WildMultiAsset)

Specify the filter as the given WildMultiAsset wildcard.

Implementations§

impl MultiAssetFilter

source

pub fn matches(&self, inner: &MultiAsset) -> bool

Returns true if inner would be matched by self.

Note that for Counted variants of wildcards, then it will disregard the count except for always returning false when equal to 0.

source

pub fn reanchor( &mut self, target: &MultiLocation, context: Junctions ) -> Result<(), ()>

Mutate the location of the asset identifier if concrete, giving it the same location relative to a target context. The local context is provided as context.

source

pub fn count(&self) -> Option<u32>

Maximum count of assets it is possible to match, if known.

source

pub fn limit(&self) -> Option<u32>

Explicit limit placed on the number of items, if any.
