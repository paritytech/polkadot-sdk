Module polkadot_sdk_frame::prelude

source ·
Expand description

The main prelude of FRAME.

This prelude should almost always be the first line of code in any pallet or runtime.

use polkadot_sdk_frame::prelude::*;

// rest of your pallet..
mod pallet {}
