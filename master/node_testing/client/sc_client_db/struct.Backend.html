pub struct Backend<Block>
where
    Block: Block,
{ /* private fields */ }
Expand description

Disk backend.

Disk backend keeps data in a key-value store. In archive mode, trie nodes are kept from all blocks. Otherwise, trie nodes are kept only from some recent blocks.
