The miner. The miner code for the EPMB pallet.

It is broadly consisted of two main types:

  • crate::unsigned::miner::BaseMiner, which is more generic, needs parameterization via crate::unsigned::miner::MinerConfig, and can be used by an external implementation.
  • crate::unsigned::miner::OffchainWorkerMiner, which is more opinionated, and is used by this pallet via the offchain_worker hook to also mine solutions during the Phase::Unsigned.
